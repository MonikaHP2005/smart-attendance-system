# Smart Attendance System for NGO Programs

## Project Description
A web-based attendance system using QR codes to streamline event management for NGOs.

## Tech Stack
* Frontend: React (Vite) + Tailwind CSS
* Backend: Node.js + Express
* Database: MySQL

## Setup Instructions
1. Clone the repository.
2. Run `npm install` in the root folder.
3. Run `cd server && npm install` and `cd ../client && npm install`.
4. Create a `.env` file in `/server` (copy from `.env.example`).
5. Import `database.sql` into your local MySQL.
6. Run `npm run dev` to start the application.
